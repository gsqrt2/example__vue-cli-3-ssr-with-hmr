# vue-cli-3-ssr (Beta)

Add ssr for vue-cli-3 app without using third paty plugins

## Running and Building

``` bash
# install dependencies
yarn install

# build all
yarn build

# build client only
yarn build:client

# build server only
yarn build:server

# Run ssr server in production mode
yarn start

# Serve with hot relod (Beta).
yarn serve

# Serve with hot relod only client
yarn serve:client

# Run lint
yarn lint
```
